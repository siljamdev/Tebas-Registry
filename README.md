# Tebas Registry
Official template & plugin registry for [Tebas](https://github.com/siljamdev/Tebas)

## Contributing
Before making a PR, execute `build.sh` with the latest tebas version installed and make sure all templates and plugins succesfully build

---

## Available Templates
- [tbplg](./templates/tbplg): Template that helps you create Tebas plugins
- [tbtem](./templates/tbtem): Template that helps you create Tebas templates

## Available Plugins
- [git](./plugins/git): Plugin that handles most git operations
- [license](./plugins/license): Plugin that adds a LICENSE to your project
- [projxp](./plugins/projxp): Project resources explorer
- [sharedxp](./plugins/sharedxp): Shared resources explorer
- [stats](./plugins/stats): Plugin that handles lines of code
- [unirun](./plugins/unirun): Universal run plugin
